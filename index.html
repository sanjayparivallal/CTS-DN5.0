🔧 DevTools Debugging Guide

Open Chrome DevTools with F12 and try the following:

  1. Elements panel – Right-click any element and select Inspect. Try changing the background-color of #welcomeBanner live.
  2. Console panel – View all console.log messages printed by this page (form submit, video events, geolocation results).
  3. Sources panel – Open the page source, find the script block, and add a breakpoint on any line to inspect variable values on reload.
  4. Application panel – Go to Storage then Local Storage to see the preferredEvent key saved by Task 8.
  5. Network panel – Reload the page and watch image and video requests load in real time.
